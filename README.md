# DevilScript
![alt tag](http://i.imgur.com/47IHNwH.png)

##### Variables
`a = 5` Creates a variable named "a" with value "5"


##### Operations
* `+` : `10 + 5`  is equal to  `15`
* `-` : `10 - 5`  is equal to  `5`
* `*` : `10 * 5`  is equal to  `50`
* `/` : `10 / 5`  is equal to  `2`
* `++` : `10 ++ 5`  is equal to  `20`
* `--` : `10 -- 5`  is equal to  `0`
